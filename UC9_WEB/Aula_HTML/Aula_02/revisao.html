Sobre

Esta revisão tem como objetivo recapitular os principais conceitos aprendidos na aula anterior, garantindo que todos estejam preparados para avançar para os próximos tópicos.

Tags importentes

  • <html>: A tag raiz que envolve todo o conteúdo da página.
  • <head>: Contém metadados, como o título da página e links para arquivos CSS.
  • <body>: Contém o conteúdo visível da página, como texto, imagens e links.
  • <h1> a <h6>: Tags de cabeçalho usadas para organizar o conteúdo em seções hierárquicas.
  • <p>: Tag de parágrafo usada para agrupar blocos de texto.
  • <img>: Tag usada para inserir imagens na página.
  • <a>: Tag usada para criar links para outras páginas ou recursos.
  • <strong>: Tag usada para destacar texto em negrito, indicando importância.
  • <em>: Tag usada para enfatizar texto, geralmente exibido em itálico.
  • <ul>: Tag usada para criar listas não ordenadas.
  • <ol>: Tag usada para criar listas ordenadas.
  • <li>: Tag usada para definir itens de uma lista.
  • <footer>: Tag usada para definir o rodapé de uma seção ou página.
  • <header>: Tag usada para definir o cabeçalho de uma seção ou página.
  • <nav>: Tag usada para definir a navegação de uma seção ou página.
  • <main>: Tag usada para definir o conteúdo principal de uma seção ou página.
  • <section>: Tag usada para definir uma seção de conteúdo na página.
  • <aside>: Tag usada para definir conteúdo relacionado, mas secundário, ao conteúdo principal.
