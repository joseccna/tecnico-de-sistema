#CodeSphere - Landing Page Responsiva

Projeto de uma landing page responsiva na plataforma CodeSphere, desenvolvida com HTML5 e CSS3 utilizando Mobili First.

##Sobre o Projeto

Este projeto tem como objetivo aplicar conceitos de deselvolvimentos front-end, como HTML semântico, CSS responsivo e boas práticas de layout.

##Conceitos Estudados

###HTML Semântico

HTML semântico é o uso de tags que possuem significado, como <header>, <nav>, <main> e <footer>.
Diferente da <div>, que não tem significado, essas tags ajudam a organizar melhor o conteúdo.
Isso melhora a acessibilidade e também o SEO, pois mecanismos de busca conseguem enterder melhor a estrutura da página.

###Mobili First

Mobili First é uma abordagem onde o desenvolvimento começa pelas telas menores(celuares) e depois é adaptado para telas maiores.
O CSS base é feito para mobile, e media queries são usadas para expandir o layout.
Isso garente melhor performance e foco no conteúdo essencial.

###CSS Custom Properties (Variáveis CSS)

Variáveis CSS permitem armazenar valores reutilizáveis, como cores e fontes.
Elas são declaradas no :root usando --nome-da-variavel e utilizadas com var().
Isso facilita a manutenção e evita repetição de código.

##Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox

##Responsividade

O projeto foi desenvolvido utilizando a abordagem Mobile First, com adaptação para telas maiores através de media queries com breakpoint de 768px.
